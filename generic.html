Background

Summary

While working as a controller at Egbin power plant security control room in early 2019, I was introduced to programming by my big brother. I have always been curious about software development but after reading the following books: Think python and principle of web development I was officially hooked. Not only was it intellectually stimulating but it was the first time I found myself doing something I loved.

Since being introduced to programming, I have read through as much material as one can possibly go through, attended classes, created a portfolio of applications and got satisfied as an associate python coder. I have truly discovered my passion in life.

Motivation

Just as in art, no two developers have the same thinking or perception while writing code for the same outcome. While some struggle to produce the desired outcome, to a few, it comes almost naturally, as if an epiphany hits them at the moment they sit to create programs or solve a problem.
For this reason, I have adopted these three powerful tools that keep me flying high: PERSEVERANCE, PRACTICE and PROACTIVENESS

However, there is a man who motivates me at all time. the man behind the secret of python programming language, He created the language all by himself his name is Guido Van Rosum.

This is it: "If someone created it, why can't I use it well?" With this I never wanted any potential to be left out of the python programming language unused by me. so, I always believe that there is more to wat I do..
